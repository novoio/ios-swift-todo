# A Simple iOS To-do App by Swift
> The app contains a few pages allowing users to manage their own tasks.  
> It consumes the RESTful API endpoints from [here](https://github.com/novoio/rails-api-todo).

**UI/UX**
* User Sign-up
* User Log-in
* Task Management

**CocoaPods**
* AFNetworking
* RestKit
* Sliding Menu
* NSUserDefaults
